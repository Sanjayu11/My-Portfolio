5-DOF Gesture Controlled Robotic Arm

This project features a 5-DOF robotic arm mounted on a mobile vehicle, designed to be controlled using a gesture-based glove. The glove is equipped with flex (bent) sensors and switches, which capture finger movements and user commands to control both the robotic arm and vehicle motion. It enables intuitive, wireless human-machine interaction and can be used in applications like remote handling, hazardous environment operations, or assistive robotics
